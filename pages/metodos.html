Concepto

Los metodos de cadena son aquellos que vienen pre-definidos en js y por lo tanto pueden emplearse en cualquier variable que tenga como valor un texto.

.concat()

Junta 2 o más cadenas y retorna 1.

INPUT:
let cadena = "hola";
let resultado = cadena.concat("mundo")
console.log(resultado)
OUTPUT:
'holamundo'

new String()

Manera alternativa de crear una cadena string. Se crea un objeto que aloja el string

INPUT:
let a = new String('Cadena de prueba');
console.log(a)
OUTPUT:
String {'Cadena de prueba'}

INPUT:
let a = new Number(3245);
console.log(a)
OUTPUT:
Number {3245}

INPUT:
let a = new Boolean(true);
console.log(a)
OUTPUT:
Boolean {true}

Tambien se pueden crear nuevos arrays con new Array

INPUT:
let a = new Array('1','2','3'); console.log(a); OUTPUT:
(3) ['1', '2', '3']
NOTA: Tambien se puede declarar la cantidad de elementos que tendrá especificando la cantidad así new Array(3);

.startsWith()

Si una cadena comienza con los carácteres de otra cadena devuelve true, sino false. Es decir, la primer palabra de la cadena debe coincidir con la primer palabra de la otra para retornar true.

INPUT:
let cadena = "hola";
let resultado = cadena.startsWith("hola Mundo")
console.log(resultado);
OUTPUT:
true

.endssWith()

Si una cadena termina con los carácteres de otra cadena devuelve true, sino false. Es decir, la ultima palabra de la cadena debe coincidir con la ultima palabra de la otra para retornar true.

INPUT:
let cadena = "Mundo";
let resultado = cadena.endsWith("hola Mundo")
console.log(resultado);
OUTPUT:
true

.includes()

Si una cadena puede encontrarse dentro de otra cadena devuelve true, sino false.

INPUT:
let cadena = "Hola Mundo";
let resultado = cadena.includes("Mundo")
console.log(resultado);
OUTPUT:
true

.indexOf()

Devuelve el índice del primer caracter de la cadena, si no existe devuelve -1. En caso de un array devolvera el indice de el valor, y en caso de un array devolvera el indice de la primer letra.

INPUT:
let cadena = "Hola Mundo";
let resultado = cadena.indexOf("Mundo")
console.log(resultado);
OUTPUT:
6
Nota: Devuelve 6 porque "M" es la sexta letra dentro de la primera cadena.

Indices

Los indices devueltos de un string corresponderan al order de los caracteres, es decir. Si no es un array, el indice devolvera la letra correspondiente al "puesto"(el indice).

Ej:
INPUT: let a = 'Hola';
console.log(a[2]);
OUTPUT:
"l"

.lastIndexOf()

Devuelve el último indice del primer caracter de la cadena, si no existe devuelve -1. Es decir, si hay muchas palabras, o multiples caracteres iguales, devolvera el indice del último caracter que corresponda.

INPUT:
let cadena = "Mundo Mundo";
let resultado = cadena.lastIndexOf("Mundo")
console.log(resultado);
OUTPUT:
7

.padStart()

El método padStart() rellena la cadena actual con una cadena dada (repetida eventualmente) de modo que la cadena resultante alcance una longitud dada. El relleno es aplicado desde el inicio (izquierda) de la cadena actual. Rellena el espacio con los caracteres deseados.
Nota: Este metodo todavía no ha sido implementado.

INPUT:
let cadena = "Hola Mundo";
let resultado = cadena.padStart(14,'xy')
console.log(resultado);
OUTPUT:
'xyxyHola Mundo'

.padEnd()

El método padStart() rellena la cadena actual con una cadena dada (repetida eventualmente) de modo que la cadena resultante alcance una longitud dada. El relleno es aplicado desde el final (derecha) de la cadena actual. Rellena el espacio con los caracteres deseados.
Nota: Este metodo todavía no ha sido implementado.

INPUT:
let cadena = "Hola Mundo";
let resultado = cadena.padStart(14,'xy')
console.log(resultado);
OUTPUT:
'Hola Mundoxyxy'

.repeat()

Devuelve la misma cadena de texto pere repetida la cantidad de veces elegida.

INPUT:
let cadena = "123 ";
let resultado = cadena.repeat(2)
console.log(resultado);
OUTPUT:
'123 123 '

.split()

Divide la cadena como le pidamos. Devuelve un array.

INPUT:
let cadena = "Hola,Mundo";
let resultado = cadena.split(',')
console.log(resultado[0]);
OUTPUT:
'Hola'

.substring()

El substring() método devuelve un subconjunto de un objeto String. Recibe por parámetro dos números el primero será el indice del carácter donde empieza el "recorte" y el segundo el indice donde termina (este no será incluido).

INPUT:
let cadena = "Hola Mundo";
let resultado = cadena.substring(0,2)
console.log(resultado);
OUTPUT:
'Ho'

.toLowerCase()

Convierte una cadena a minúscula.

INPUT:
let cadena = "Hola Mundo";
let resultado = cadena.toLowerCase();
console.log(resultado);
OUTPUT:
'hola mundo'

.toUpperCase()

Convierte una cadena a mayúscula.

INPUT:
let cadena = "Hola Mundo";
let resultado = cadena.toUpperCase();
console.log(resultado);
OUTPUT:
'HOLA MUNDO'

.toString()

Devuelve una cadena que representa al objeto. Es decir, cambia el valor a una cadena de texto.

INPUT:
let cadena = true;
let resultado = cadena.toString();
console.log(resultado);
OUTPUT:
'true'
Nota: También podrá convertir arrays a string seperados por coma.

.length

Devuelve la longitud de una cadena de texto. Esta propiedad devuelve el número de caracteres de una cadena. UTF-16, el formato usado por JavaScript, usa 16-bit para representar los caracteres más comunes, pero necesita usar dos caracteres para otros menos usados, así que es posible que el valor devuelto por length no corresponda al número de caracteres de la cadena.

INPUT:
let cadena = 'hola';
let resultado = cadena.length;
console.log(resultado);
OUTPUT:
4
Nota: También podrá convertir arrays a string seperados por coma.

.trim()

El método trim( ) elimina los espacios en blanco en ambos extremos del string. Los espacios en blanco en este contexto, son todos los caracteres sin contenido (espacio, tabulación, etc.) y todos los caracteres de nuevas lineas (LF,CR,etc.).

INPUT:
let cadena = '    hola    ';
let resultado = cadena.trim();
console.log(resultado);
OUTPUT:
'hola'

.trimStart()

Elimina los espacios del principio

INPUT:
let cadena = '    hola    ';
let resultado = cadena.trimStart();
console.log(resultado);
OUTPUT:
'hola    '

.trimEnd()

Elimina los espacios del final

INPUT:
let cadena = '    hola    ';
let resultado = cadena.trimEnd();
console.log(resultado);
OUTPUT:
'    hola'

.valueOf()

El método valueOf() retorna el valor primitivo del objeto especificado.

INPUT:
let cadena = new String('Texto');
let resultado = cadena.valueOf();
console.log(resultado);
OUTPUT:
'Texto'

.shift()

Elimina el primer elemento de un array y lo devuelve. METODO TRANSFORMADOR. Si imprimo sobre una nueva variable devolvera el elemento borrado.

INPUT:
let letras = ['a', 'b', 'c'];
let resultado = letras.shift();
console.log(letras);
console.log(resultado);
OUTPUT:
(2)['b', 'c']
'a'

.pop()

Elimina el último elemento de un array y lo devuelve. METODO TRANSFORMADOR. Si imprimo sobre una nueva variable devolvera el elemento borrado.

INPUT:
let letras = ['a', 'b', 'c'];
let resultado = letras.pop();
console.log(letras);
console.log(resultado);
OUTPUT:
(2)['a', 'b']
'c'

.unshift()

Agrega un elemento al principio del array y lo devuelve. METODO TRANSFORMADOR. Si imprimo sobre una nueva variable devolvera la longitud del array.

INPUT:
let letras = ['a', 'b', 'c'];
let resultado = letras.unshift(0);
console.log(letras);
console.log(resultado);
OUTPUT:
(4)[0, 'a', b', 'c']
4

.push()

Agrega un elemento al final del array y lo devuelve. METODO TRANSFORMADOR. Si imprimo sobre una nueva variable devolvera la longitud del array.

INPUT:
let letras = ['a', 'b', 'c'];
let resultado = letras.unshift(0);
console.log(letras);
console.log(resultado);
OUTPUT:
(4)['a', b', 'c', 0]
4

.reverse()

Invierte el orden los elementos de un array

INPUT:
let nums = [1, 2, 3];
nums.reverse();
console.log(nums);
OUTPUT:
(3)[3, 2, 1]

.sort()

El método sort() ordena los elementos de un arreglo (array) localmente y devuelve el arreglo ordenado. Es decir lo ordena alfabeticamente y de menor a mayor. METODO MODIFICADOR.

INPUT:
let nums = [7, 1, 5];
nums.sort();
console.log(nums);
OUTPUT:
(3)[1, 5, 7]

.splice()

El método splice() cambia el contenido de un array eliminando elementos existentes y/o agregando nuevos elementos. El primer parametro que contendra splice será el del indice en el cual comenzara el corte y el segundo la cantidad de elementos que eliminara hacia la derecha empezando a contar desde el elemento del indice eleegido. El tercero correspondera al elemento que quiera agregar.

INPUT:
let nums = ['a', 'b', 'c'];
nums.splice(1,2,'Hola Mundo');
console.log(nums);
OUTPUT:
(2)['a', 'Hola Mundo']
Nota: Este metodo puede ser empleado para agregar elementos especificando como argumento 0,0 siendo el primero argumento el indice que le será asignado al elemento o -1 para que el elemento sea colocado al final.

.join()

El método join() une todos los elementos de una matriz (o un objeto similar a una matriz) en una cadena y devuelve esta cadena. METODO ACCESORIO. Transforma el array en cadena de texto permitiendonos especificar el divisor entre valores que queramos.

INPUT:
let letras = ['a', 'b', 'c'];
let resultado = letras.join('-');
console.log(nums);
OUTPUT:
'a-b-c'

.slice()

Devuelve una parte del array teniendo por parametro un inicio y un fin(fin no incluido)

INPUT:
let letras = ['a', 'b', 'c'];
let resultado = letras.slice(0,2);
console.log(resultado);
OUTPUT:
(2) ['a', 'b']
Nota: para copiar el array completo, deberé introducir como argumento -1 en el segundo parametro.

Otros

.toString() Convierte el array en un string.

.indexOf() Muestra el primer indice que coincida con la busqueda. Busca la cadena de texto entera

.lastIndexOf() Muestra el ultimo indice que coincida con la busqueda. Busca la cadena de texto entera

.includes() Devuelve true si existe tal inclusion en el array.

.length() Devuelve cuantos elementos tiene el array. En caso de strings devolvera la cantidad de los carácteres

.filter()

Crea un nuevo array con todos los elementos que cumplan con la condición.

La condición debe arrojar un valor boolean que, en caso de ser true, devolvera dentro del array el valor que cumpla con la condición dentro de la iteración.

INPUT:
let letras = ['a', 'b', 'c'];
let resultado = letras.filter(letra => console.log(letra));
let resultado2 = letras.filter(letra => letra == 'c');
console.log(resultado2);
OUTPUT:
'a'
'b'
'c'
'c'
Nota: En vez de una función flecha, el argumento que recibe filter puede ser una función común. Nota 2: Ahora resultado tiene el mismo valor que letras. Nota 3: Ahora resultado2 tiene solo 'c', que es el único valor del array que cumple con la codición.

.forEach()

El método forEach() ejecuta la función indicada una vez por cada elemento del array.

INPUT:
const array1 = ['a', 'b', 'c'];
array1.forEach(element => console.log(element));
OUTPUT:
'a'
'b'
'c'

setInterval

Establece un bucle en el cual se ejecuta una funcion continuamente cada la cantidad de milisegundos establecida, siendo el primer parámetro el funcion y el segundo los minisegundos, cuando se escriba la funcion como argumento deberá ir sin parentesis.

Ej:
const alertar = () => { alert('ALERTA!') }
setInterval(alertar, 3000);

date()

Crea un objeto que aloja la fecha y hora actual siempre que este acompañado de la palabra new EJ let a = new Date();

Las fechas en JavaScript son calculadas en millisegundos desde el 1 de enero de 1970, y cuenta los meses desde 0 (enero) a 11 (diciembre)
