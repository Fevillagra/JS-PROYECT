Consola

La consola JavaScript de nuestro navegador sirve para ver los logs que lanzamos en nuestros scripts y como intérprete JavaScript desde el que lanzar pequeños comandos y ver sus efectos en la web.

console.log()

Muestra un mensaje en la consola web (o del intérprete JavaScript).

console.assert()

NO ESTANDAR. Aparece un mensaje de error en la consola si la afirmación es falsa, si la afirmación es verdadera, no aparecera nada.

INPUT:
console.assert(4 > 5);
OUTOUT:
Assertion failed console.assert

cosole.clear() o clear()

Limpía la consola.

console.error()

Muestra un mensaje de error en la consola.

INPUT:
console.error('Error!!!');
OUTPUT:
Error!!!

console.info()

Emite un mensaje informativo a la Consola Web. En Firefox y Chrome, se muestra un pequeño ícono "i" junto a estos elementos en el registro de la Consola Web.

INPUT:
console.info('Éxito!');
OUTPUT:
'Éxito!'

console.table()

Muestra datos tabulares como una tabla.
Esta función toma un argumento obligatorio: data, que debe ser un array o un objeto, y un parámetro adicional: columns.

INPUT:
console.table([1,2,3,4,5]);
OUTPUT: (Será una tabla con el index y su respectivo valor)

console.warn()

Imprime un mensaje de advertencia en la Consola Web.

INPUT:
console.warn('Cuidado');
OUTPUT:
Cuidado

console.dir()

NO ESTANDAR. El método dir() de Console despliega una lista interactiva de las propiedades del objeto JavaScript especificado. El resultado es presentado como un listado jerárquico con triángulos de despliegue que te dejan ver los contenidos de los objetos hijos.

console.count()

Registra el número de veces que se llama a count(). Esta función toma como argumento opcional una etiqueta.

INPUT:
console.count();
console.count();
console.countReset();
console.count();
OUTPUT:
default: 1
default: 2
default: 1

console.group();

El console.group()método crea un nuevo grupo en línea en el registro de la consola web, lo que hace que los mensajes de la consola subsiguientes tengan una sangría de un nivel adicional, hasta console.groupEnd()que se llame.

Se desplegara un area en el cual se puede escribir comandos que serán registrados dentro de console.group. El nombre del grupo es el parámetro que recibe. Para eliminar el grupo: console.groupEnd();

console.groupCollapsed(); Como group, pero estará cerrado.

console.time()

El console.time()método inicia un temporizador que puede usar para rastrear cuánto tiempo lleva una operación. Le da a cada temporizador un nombre único y puede tener hasta 10,000 temporizadores ejecutándose en una página determinada. Cuando llame console.timeEnd()con el mismo nombre, el navegador mostrará el tiempo, en milisegundos, transcurrido desde que se inició el temporizador. console.timeLog() Mostrará el tiempo transcurrio desde la ejecución de console.time, pero no lo parará.

Estilos en consola

Para cambiar estilos será necesario dos parámetros, el primero será el mensaje que quiero imprimir en consola con el sufijo %c el segundo los estilos en lenguaje css.

INPUT:
console.log('%chola','color:red;brackground:black;')
