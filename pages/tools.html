Herramientas para desarrolladores

Es un conjunto de herramientas para desarrolladores web integradas directamente en el navegador. Cuenta con un menu para elegir como será mostrada la consola, la opcion de vista mobile y un selector de elementos. En la ventana de vista mobile en el menu superior derecho hay una opcion para ver mediaQuerys. Se pueden tomar screenshots. ctrl + shift + p Accede al panel menu de screenshoots.

Para seleccionar un elemento en especifico se le da click al elemento seguido de inspeccionar. Para entrar a la consola f12

Pestaña ELEMENTS o elementos

La pestaña elements muestra el codigo html en la parte superior.

Click secundario

COPY: Con click secundario se copiar y cortar el elemento, outerHTML selector, estilos, xpad(alojamiento), fullxpad(alojamiento + informacion),

Force state: Fuerza una pseudoclase a un elemento que no tiene la pseudoclase activa en ese momento. En la ventana secundaria de estilos a la derecha del cuadro de "filtrar" tambien se podrá selectionar opciones de force state.

Se puede crear eliminar y modificar etiquetas. Con ctrl f puedo encontrar palabras clave dentro de elements.

Pestaña secundaria styles

En esta pestaña se pueden ver y modificar los estilos del elemento seleccionado. Puedo usar en cualquier elemento que tenga color el gotero para elegir exactamente el color de la página que estoy buscando.
A la derecha aparece el archivo del estilo y la linea del selector en el archivo css.

Pestaña RESOURCES o fuentes

Permite trabajar directamente con los archivos que están adentro. (archivos de la página). Puedo modificar los archivos y guardar los cambios con cntrl + s. Modifica los archivos en tiempo real. En el menu izquierdo hay la opcion de open file para trabajar con cualquier archivo.

Pestaña secundaria snippets o fragmentos

Son fragmentos de codigo que puedo guardar en el navegador para modificar el codigo con posterioridad al igual con la consola con la diferencia de que no se borrara si este es guardado. Si recargo la página, para activar el snippet será necesario seleccionarlo e introducir ctrl +

Pestaña NETWORK o red

Es una pestaña que muestra en la parte inferior los archivos y nos permite agregar opciones. Muestra el initiator que es el archivo que lo inicializa, el tipo de documento, estado, el tamaño, el tiempo que tardo en cargar y el waterfall que muestra la linea en donde se inicializo.
El cuadro el centro muestra en que segundo empezo a cargar el archivo y en que segundo termino.
En la parte inferior muestra cuanto tiempo tardo en cargarse toda la pagina. Las transferencias, las solicitudes, el tamaño de los archivos totales el tiempo finalizado y el tiempo del dom cargando.
Puedo darle click secundario y bloquear url request para que no cargue luego de actualizar la pagina. Se puede tambien bloquear el dominio entero. Para borrar el cache tambien es con click secundario. Tambien se pueden borrar las cookies.
