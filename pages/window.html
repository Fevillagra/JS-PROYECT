Window

El objeto window representa la ventana que contiene un documento DOM; la propiedad document apunta al DOM document cargado en esa ventana. El objeto window al que pertenece un documento puede ser obtenido usando la propiedad document.defaultView. Hereda las propiedades de EventTarget.

window.open()

Abre una ventana nueva

EJ:
window.open('https://google.com');

.close()

Cierra una ventana abierta por window.open().

EJ:
let ventana = window.open('https://google.com');
ventana.close();
Nota: La ventana se cerrara, al ejecutarse todo el código abrira e inmediatamente cerrara sin que nos percatemos.

window.closed

Retorna true si la ventana referenciada está cerrada.

INPUT:
let ventana = window.open('https://google.com');
ventana.close();
ventana.closed;
OUTPUT:
true

window.stop()

Detiene la carga de recursos en el contexto de la navegación actual. Similar a apretar el boton de cruz cuando se está cargado la página.

Ej:
window.stop();

window.alert()

Muestra un mensaje de alerta con el contenido especificado y un boton aceptar

EJ:
alert('Bienvenido');

window.print()

Abre un cuadro de dialogo para imprimir el documento actual.

EJ:
print();

window.prompt()

Abre un cuadro de dialogo con un mensaje que solicita al usuario un dato (string).

EJ:
prompt('Ingrese nombre');

window.confirm()

Abre un mensaje de dialogo con dos botones aceptar y cancelar. Si el usuario apreta aceptar devuelve true, si elije cancelar, false.

EJ:
confirm('Desea salir?');

window.screen

Devuelve una referencia al objeto de pantalla asociado con la ventana. El objeto screen, que implementa la interfaz screen, es un objeto especial para inspeccionar las propiedades de la pantalla en la que se representa la ventana actual.

EJ:
window.screen

screen.availHeight

Especifica la altura de la pantalla, en píxeles, menos las funciones de interfaz de usuario permanentes o semipermanentes que son mostradas por el sistema operativo, como la barra de tareas de Windows.

EJ:
console.log(window.screen.availHeight);

screen.availWidth

Devuelve la cantidad de espacio horizontal disponible en pixeles para la ventana.

EJ:
console.log(window.screen.availWidth);

window.screenLeft

La propiedad Window.screenLeft de solo lectura devuelve la distancia horizontal, en píxeles CSS, desde el borde izquierdo de la ventana gráfica del navegador del usuario hasta el lado izquierdo de la pantalla. Es decir la distancia entre el borde de la ventana izquierdo al final de la pantalla

EJ:
console.log(window.screenLeft)

window.screenTop

La propiedad Window.screenTop de solo lectura devuelve la distancia vertical, en píxeles CSS, desde el borde superior de la ventana gráfica del navegador del usuario hasta el lado superior de la pantalla. Es decir la distancia entre el borde de la ventana superior al final de la pantalla.

EJ:
console.log(window.screenTop)

window.scrollX

Devuelve el número de pixeles que el documento se desplaza horizontalmente actualmente

EJ:
console.log(window.scrollX);

Devuelve el número de pixeles que el documento se desplaza verticualmente actualmente

EJ:
console.log(window.scrollY);

window.scroll()

Desplaza la ventana a un lugar particular en el documento. El primer parámetro es la coordenada x y el segundo la coordenada y. Los dos en pixeles. Los valores que recibe son tipo number.
Desplazamiento relativo: scrollBy, scrollByLines, scrollByPages.

EJ:
window.scroll(0,300);

window.minimize()

Minimiza la ventana. OBSOLETA NO FUNCIONA.

window.resizeBy()

Cambia el tamaño de la ventana actual en una cantidad especifica. El primer parámetro es el ancho y el segundo el alto. INTERACTUA CON EL SERVIDOR

window.resizeTo()

Cambia el tamaño de la ventana actual en una cantidad especifica relativa. INTERACTUA CON EL SERVIDOR.

window.moveTo()

Mueve la pantalla. Recibe dos parametros el ancho y alto. INTERACTUA CON EL SERVIDOR.

window.moveToBy()

Mueve la pantalla relativamente. Recibe dos parametros el ancho y alto. INTERACTUA CON EL SERVIDOR.

Objetos barprop

Retorna true si los objetos son visibles: locationbar, personalbar, scrollbars, statusbar, toolbar.

INPUT:
window.locationbar.visible
OUTPUT:
true

window.location

La propiedad de sólo lectura Window.location retorna un objeto Location con información acerca de la ubicación actual del documento.

INPUT:
let ubicacion = window.location
console.log(ubicacion);
OUTPUT:
Location {ancestorOrigins: DOMStringList, href: 'http://127.0.0.1:5500/pages/window.html#window', origin: ' http://127.0.0.1:5500', protocol: 'http:', host: '127.0.0.1:5500', …}

location.href

Refleja donde está localizado el archivo.

INPUT:
let ubicacion = location.href
console.log(ubicacion);
OUTPUT:
'http://127.0.0.1:5500/pages/window.html#window'

location.hostname

Devuelve el nombre del dominio.

INPUT:
let ubicacion = location.hostname
console.log(ubicacion);
OUTPUT:
'127.0.0.1'

location.pathname

Devuelve la locacion dentro del dominio.

INPUT:
let ubicacion = location.pathname
console.log(ubicacion);
OUTPUT:
'/pages/window.html'

location.protocol

Devuelve el protocolo

INPUT:
let ubicacion = location.protocol
console.log(ubicacion);
OUTPUT:
'http:'

location.assign()

Carga un nuevo documento.

location.assign('http://www.google.com');
