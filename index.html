¿Qué es JavaScript?

JavaScript (abreviado comúnmente JS) es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.
Se utiliza principalmente del lado del cliente, implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas y JavaScript del lado del servidor (Server-side JavaScript o SSJS). Su uso en aplicaciones externas a la web, por ejemplo en documentos PDF, aplicaciones de escritorio (mayoritariamente widgets) es también significativo.

Orientado a objetos

La Programación Orientada a Objetos (POO, en español; OOP, según sus siglas en inglés) es un paradigma de programación que parte del concepto de "objetos" como base, los cuales contienen información en forma de campos (a veces también referidos como atributos o propiedades) y código en forma de métodos.
Los objetos son capaces de interactuar y modificar los valores contenidos en sus campos o atributos (estado) a través de sus métodos (comportamiento).

Basado en prototipos

Programación basada en prototipos, también llamada programación basado en instancias o classless, es un estilo de programación orientada a objetos en el cual los objetos no son creados mediante la instanciación de clases sino mediante la clonación de otros objetos o mediante la escritura de código por parte del programador. De esta forma los objetos ya existentes pueden servir de prototipos para los que el programador necesite crear.

Imperativo

Los lenguajes de programación imperativa son como unas instrucciones paso a paso (cómo) redactadas para el ordenador. Describen de forma explícita qué pasos deben llevarse a cabo y en qué secuencia para alcanzar finalmente la solución deseada.

Débilmente tipado

Un lenguaje debilmente tipado es aquel que cuando las variables que usas para desarrollar tu programa no les tienes que asignar un tipo de variable (como String, entero entre otros).

Dinámico

Un lenguaje de programación es dinámicamente tipado si una variable puede tomar valores de distintos tipos. La mayoría de lenguajes de tipado dinámico son lenguajes interpretados, como Python o Ruby. Un lenguaje que no es dinámicamente tipado se dice que es de tipado estático, o estáticamente tipado.

Usos

El uso más común de JavaScript es escribir funciones embebidas o incluidas en páginas HTML y que interactúan con el Document Object Model (DOM o Modelo de Objetos del Documento) de la página. Algunos ejemplos sencillos de este uso son:


  • Cargar nuevo contenido para la página o enviar datos al servidor a través de AJAX sin necesidad de recargar la página (por ejemplo, una red social puede permitir al usuario enviar actualizaciones de estado sin salir de la página).

  • Animación de los elementos de página, hacerlos desaparecer, cambiar su tamaño, moverlos, etc.

  • Contenido interactivo, por ejemplo, juegos y reproducción de audio y vídeo.

  • Validación de los valores de entrada de un formulario web para asegurarse de que son aceptables antes de ser enviado al servidor.

  • Transmisión de información sobre los hábitos de lectura de los usuarios y las actividades de navegación a varios sitios web. Las páginas Web con frecuencia lo hacen para hacer análisis web, seguimiento de anuncios, la personalización o para otros fines.

Otros usos

FrameWorks y librerias: Facilitan la creacion de páginas web dinamicas con Js como protagonista principal de todo el codigo involucrado del lado del FrontEnd.

NodeJs: Mediante la utilizacion de Node tambien se puede utilizar para ejecutar codigo js del lado del BackEnd.

Usos menos comunes

Inteligencia artificial, placas electronicas, mobile apps, deskop apps, etc.

Formas de enlazar JS

En Línea

Mediante el uso de una propiedad dentro de la etiqueta con el nombre del evento, similar al uso de style en css.

Ej:
<h1 onclick="alert('Hola Mundo!')"> Titulo <h1>

Como contenido en la etiqueta script

Usualmente se coloca la etiqueta script al final de la etiqueta body con el objetivo de que cargue primero la estructura de la pagina y así evitar errores. Para que este metodo funcione, la propiedad src no debera estar establecida en la etiqueta script. Suele incluirse dentro de esta etiqueta el atributo type="text/javascript" pero no es necesario.

Ej:
<script>
alert('Hola Mundo!')
<script>

Como contenido en un archivo de formato .js

Todo el codigo estara escrito en uno o varios archivos js cuya ruta será referenciada dentro de la etiqueta script dentro de la propiedad src.

Ej:
<script src="./js/index.js"> <script>
<script src="./js/carrousel.js"> <script>

Una variable es un espacio en la memoria que aloja un dato. Dicha variable contiene en su interior un valor.
Una variable es una ubicación nombrada para almacenar un valor. De esta manera se puede acceder a un valor impredecible por medio de un nombre predeterminado.

Ej:
recipiente = agua;
Siendo la primer palabra la variable y la segunda el valor.

Tipos de valores

Existen diferentes tipos de valores que pueden ser establecidos dentro de una variable: Son valores primitivos o datos primitivos ya que vienen por defecto en JS.

  • String(cadena de texto) Cadena de caracteres alfanumericos

    Ej: let a = "Cadena de texto"

  • Number(número) Numero entero o decimal.

    Ej: let a = 5.5

  • Boolean 0 o 1, traducidos en true o false.

    Ej: let a = true

    Ej: let b = false

  • Casos Especiales: undefined, null & nan.

    Por lo general, comunican que la variable no está definida o que hay un error

  • undefined Es un valor que aparece cuando la variable let no fue inicializada

  • null Es una variable vacía, este valor fue establecido intencionalmente.

  • nan Es un valor que aparece cuando realizo operaciones aritmeticas con variables que alojan tipos de valores entre si y que la operación existente entre estos da un resultado inexistente. Por ejemplo multiplicar un valor number por uno string.

Scope(alcanze)

El scope de una variable hace referencia al lugar donde va a vivir esta, o podrá ser accesible, en JavaScript tenemos varias opciones: global, local y bloque.

Declaración e inicialisación

Una variable se declara para indicarle al programa a partir de qué lugar empieza a existir, qué nombre tendrá y qué tipo de datos almacenará. La asignación de un valor inicial se llama inicialización.

Para declarar variables se usan 3 formas: var, let y const.
Al finalizar una afirmación deberá establecerse un ";" con el proposito de separar el contenido.

Ej declaración:
let
let a;
a = 5;
a = 6;

Con let se puede declarar la variable e inicializarla luego. Tambien se puede reasignar el valor. Si no es inicializada, el valor por defecto sera undefined.
Tambien puede ser declarada e inicializada al mismo tiempo:
let a = 5;


const
const b = 5;
Con const de debe declarar la variable e inicializarla al mismo tiempo para que no haya errores, a diferencia de let, el valor que recibe no puede ser reasignado.


var
var c = 3;
c = 2;
var c = 5;
Con var también se puede declarar la variable e inicializarla luego, la diferencia con let es que var puede iniciliazarse multiples veces.

Actualmente var esta en desuso ya que let, a diferencia de var, límita el scope de la variable desde donde fue inicializada. Es decir, una variable declarada con let podrá ser usada en todos los bloques inferiores y en el bloque en que fue inicializada a diferencia de var que puede ser usada de manera global.

Crear multiples variables

Se pueden declarar multiples variables al mismo tiempo con la siguiente sintaxis:


Ejemplo 1:

let num1, num2, num3;
num1 = 10;
num2 = 20;
num3 = 30;


Ejemplo 2:

let num1 = 10, num2 = 20;

Hoisting

El concepto de Hoisting fue pensado como una manera general de referirse a cómo funcionan los contextos de ejecución en JavaScript (específicamente las fases de creación y ejecución)
Ej:
let a;
alert(a);
a = 5;
Nota: En este caso cuando se alerte la variable a mostrara el valor undefined ya que la inicialización ocurre despues, en el caso de funciones, esto no ocurre ya que las funciones pueden ser llamadas antes de ser establecidas. El hoisting se vera en profundidad en sectores posteriores a este.

alert()

El metodo alert muestra un dialogo con un mensaje opcional

prompt()

El método prompt() muestra un diálogo con mensaje opcional, que solicita al usuario que introduzca un texto. Dicho valor puede ser almacenado dentro de una variable.
Ej:
let a = prompt('Introduzca una palabra'); Nota: Si el usuario hace ingresa cancelar el prompt devolvera null.

confirm()

Un cuadro de confirmación se usa a menudo para que el usuario verifique o acepte algo. Cuando aparece un cuadro de confirmación, el usuario debe hacer clic en Aceptar o Cancelar para continuar. Si el usuario hace clic en Aceptar, el cuadro devuelve verdadero . Si el usuario hace clic en Cancelar, el cuadro devuelve falso.

EJ:
let resultado = confirm('Deseas irte de la página?');
if(resultado) {
    alert('Gracias!');
} else {
    alert('Gracias por contar con nosotros.');
}

parseInt()

Convierte (parsea) un argumento de tipo cadena y devuelve un entero de la base especificada.

document.write()

Escribe sobre la pagina un texto

Operadores

Un operador de asignación asigna un valor al operando de la izquierda basado en el valor operando de la derecha.

Asignación de resta x -= y x = x - y Resta
Asignación de multiplicación x *= y x = x * y Multiplicacion

Nota: En el siguiente ejemplo se muetra la manera correcta de guardar el resultado en una variable:
let a = 5 a++ resultado = a Se hace en este orden ya que si se guarda a++ en resultado no daría como resultado 6.

Operadores arítmeticos

Los operadores arítmeticos toman valores númericos(ya sean literales o variables) como sus operandos y retornan un valor numerico único.

Suma (+). Suma dos números: 3 + 2 = 5 . Si en lugar de números se suman cadenas, como por ejemplo "A" + "B" = "AB" se produce una concatenación. Si alguno de los elementos a concatenar no es una cadena de texto, queda convertido a cadena: "AB" + 123 = "AB123" .

Resta (-). Halla la diferencia entre dos números. Ejemplo A: 3-2 = 1 . Ejemplo B: (-1)-(-2) = 1 .

Incrementos y decrementos (++ --). Incrementa o decrementa el valor de una variable numérica en una unidad. No puede aplicarse a un literal. Pueden darse dos casos:
var A, B;
B = 2;
A = ++B;
A = B++;
nota: El segundo caso es más común.

Operadores de comparación

Los operadores de comparación comparan dos expresiones y devuelven un valor boolean que representa la relación de sus valores.


Nota: Con los operadores iguales en el caso de el valor 1 y "1" sera true no diferencia el tipo de dato , con los operadores estrictamente iguales === será false que si diferencia el tipo de dato.

Operadores lógicos

Los operadores lógicos devuleven un resultado a partir de que se cumpla (o no) una condición, su resultado es booleano, y sus operandos son valores lógicos o asimilables a ellos.


Nota: Primero resolvera los operadores que den como resultado un boolean y luego los operadores logicos, siempre priorizando los parentesis en ambos casos y de izquierda a derecha.

Concatenación

La concatenación es la unión de strings. Para unirlas basta con sumar dos o mas variables que alojen valores string.

Ej 1:
let a = 'Hola';
let b = 'Mundo!';
let c = a + b;
console.log(c);
OUTPUT:
'Hola Mundo!'
Ej 2:
let a = 1;
let b = 2;
let c = '' + a + b;
console.log(c)
OUTPUT:
'12' Nota: Si JS detecta algún string automaticamente todos los valores serán considerados strins.

Función concat()

El método concat() se usa para unir dos o más arrays. Este método no cambia los arrays existentes, sino que devuelve un nuevo array.

Ej:
let a = 13;
let b = 25;
let c = a.concat(b);
console.log(c);
OUTPUT:
'1325'

Camel Case

Camel case (estilizado como camelCase) o letra de caja camello es el nombre informal que se da en inglés a un estilo de escritura que se aplica a frases o palabras compuestas. El nombre se debe a que las mayúsculas a lo largo de una palabra en camelCase se asemejan a las jorobas de un camello. El nombre camelCase se podría traducir como Mayúsculas/Minúsculas Camello.

Dentro de JavaScript es una buena práctica empezar la primer palabra que compone el nombre entero de una variable, o función con minúscula y en caso de que este compuesta por más de una palabra se usara maýuscula. Ej: let nuevoProducto = '1';. También puede observarse la implementación de camelCase dentro de JavaScript nativo. Ej: 'let a = document.getElementById(navBar)'.

Condicionales

Definiciendo Bloque

Los bloques en Javascript se delimitan mediante llaves, una de apertura ‘{‘, y otra de cierre ‘}’.

if ...else

Ejecuta una sentencia si una condición específicada es evaluada como verdadera. Si la condición es evaluada como falsa, otra sentencia puede ser ejecutada.
Estructura:
if(condicion) {
alert('Es true');
} else {
alert('Es false');
}
En la estructura anterior, si la condición es verdadera se ejecuta el primer alert, en caso contrario se ejecuta el segundo alert. No es necesario definir el else. También puede especificarse ifs adicionales.

Ej 1:
if(numero > 10) {
alert('Es mayor a 10');
} else if(numero > 5) {
alert('Es mayor a 5');
} else {
alert('No es mayor a 5');
}

Basta con que una de las dos primeras sea verdadera para que el resto o la última respectivamente no se ejecute.

Comentado
